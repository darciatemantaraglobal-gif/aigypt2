## DESIGN SYSTEM — DARK LUXURY

### Warna
```css
--bg-void: #060608          /* hitam pekat — background utama */
--bg-deep: #0A0A0F          /* hitam sekunder */
--bg-elevated: #101018      /* card surface */
--bg-glass: rgba(20,20,30,0.5) /* glassmorphism */

--purple-core: #7C3AED      /* ungu utama */
--purple-bright: #A855F7    /* ungu terang */
--purple-glow: #C084FC      /* glow accent */
--purple-deep: #4C1D95      /* ungu gelap untuk depth */

--text-primary: #FAFAFA     /* putih hampir murni */
--text-secondary: #A1A1AA   /* abu terang */
--text-muted: #52525B       /* abu gelap untuk hint */

--border-hairline: rgba(255,255,255,0.06)  /* border sangat halus */
--border-glow: rgba(124,58,237,0.3)        /* border ungu glow */

--gold-accent: #D4AF37      /* opsional: emas halus untuk sentuhan luxury */
```

### Tipografi
- Display besar: **Space Grotesk** atau **Clash Display** (kalau bisa) — untuk headline besar, weight 500-600
- Body: **Inter** — clean, weight 300-400 untuk kesan ringan dan elegan
- Mono accent: **JetBrains Mono** — untuk label kecil, angka, badge

Aturan tipografi luxury:
- Headline BESAR tapi weight tidak terlalu tebal (500-600, bukan 700+)
- Letter-spacing sedikit negatif pada headline besar (`tracking-tight`)
- Line-height longgar pada body untuk kesan napas
- Banyak ukuran teks kecil dengan letter-spacing lebar (`tracking-widest`) untuk label — ini signature luxury

### Prinsip Visual
1. **Ruang adalah mewah** — whitespace (atau "blackspace") yang berlimpah. Jangan padat.
2. **Glow halus, bukan neon** — efek cahaya ungu yang lembut di balik elemen kunci, blur tinggi, opacity rendah
3. **Gradien gelap** — radial gradient ungu sangat halus di sudut-sudut, hampir tidak terlihat
4. **Border hairline** — garis pemisah setipis mungkin (1px dengan opacity rendah)
5. **Glassmorphism selektif** — card dengan backdrop-blur dan transparansi untuk kesan kaca premium
6. **Animasi reveal halus** — elemen muncul perlahan saat scroll (fade + slight rise), tidak pernah tergesa
7. **Noise texture sangat halus** (opsional) — grain tipis di background untuk kesan filmis
8. **Tidak ada emoji di landing** — gunakan ikon garis tipis (Lucide) atau tidak sama sekali

## DETAIL ANIMASI & POLISH

1. **Scroll reveal**: Setiap section dan elemen kunci muncul dengan fade + rise halus (Framer Motion `whileInView`), durasi ~600ms, easing lembut. Stagger pada grup elemen.
2. **Hero glow**: Radial gradient ungu di belakang headline yang "bernapas" perlahan (animasi opacity sangat halus, loop lambat ~6s).
3. **Hover pada card**: border hairline berubah jadi border-glow ungu, sangat halus.
4. **Tombol primer**: glow ungu lembut yang sedikit menguat saat hover.
5. **Garis penghubung di Section Perjalanan**: menyala dari atas ke bawah saat di-scroll (animasi draw).
6. **Angka mono (01, 02, ...)**: besar, elegan, warna ungu muted.
7. **Cursor-aware glow** (opsional, desktop): glow ungu sangat halus mengikuti kursor di hero.
8. **Noise/grain overlay** (opsional): texture grain sangat tipis di seluruh halaman untuk kesan filmis premium.
9. **Smooth scroll** antar section.
10. Semua animasi hormati `prefers-reduced-motion`.
