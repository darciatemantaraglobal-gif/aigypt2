### 3. WORKFLOW SECTION (tampil di atas, sebelum grid tools)
Heading: `Workflow Terbaik: Dari Ide ke Produk`

Tampilkan visual flow horizontal (atau vertical di mobile) dengan 5 tahap:
```
Research → Plan → Execute → Review → Ship
```
Setiap tahap adalah card kecil dengan judul dan 1 kalimat deskripsi:
- **Research** — Pahami masalah dan validasi idenya
- **Plan** — Tulis PRD sederhana + bikin DESIGN.md
- **Execute** — Bangun bertahap (jangan one-shot!)
- **Review** — Cek tiap hasil sebelum lanjut
- **Ship** — Deploy dan iterate dari feedback nyata

Di bawah flow, tampilkan box highlight "7 Aturan Emas Vibe Coding" dalam bentuk list:
1. Vibe coding bukan one-shot — bangun bertahap
2. Tulis PRD dulu sebelum prompting
3. Pakai DESIGN.md biar UI tidak generik
4. Graduate workflow: prototype di Lovable/Bolt → serius di Cursor/Claude Code
5. Pilih satu icon library, konsisten (Lucide untuk shadcn)
6. Slice vertically — bangun fitur end-to-end, bukan layer per layer
7. Selalu review — 45% kode AI punya kerentanan keamanan

## IMPLEMENTASI TEKNIS

1. **Data array**: Simpan semua tools sebagai array of objects di file terpisah `lib/toolboxData.ts`:
```typescript
export interface Tool {
  name: string
  url: string
  category: 'app-builder' | 'ai-assistant' | 'icon' | 'illustration' | 'background' | 'ui-components' | 'design-system' | 'database' | 'deploy' | 'planning'
  description: string
  pricing: 'Gratis' | 'Freemium' | 'Berbayar'
}

export const tools: Tool[] = [
  // ... semua tools dari data di atas
]
```

2. **Search & Filter**: Gunakan React `useState` untuk query pencarian dan kategori aktif. Filter array secara real-time. Jangan pakai localStorage.

3. **Kategori badge warna**: Beri warna berbeda (variasi ungu/biru/hijau gelap) per kategori agar mudah dibedakan secara visual, tapi tetap dalam tema gelap.

4. **Responsiveness**: Grid 3 kolom (desktop), 2 (tablet), 1 (mobile). Search dan filter harus nyaman dipakai di mobile.

5. **Link eksternal**: Semua link tool buka di tab baru (`target="_blank" rel="noopener noreferrer"`).

6. **Tambahkan link ke navbar**: Update komponen Navbar yang sudah ada — tambahkan link `Toolbox` (hanya tampil untuk user yang sudah login).

7. **Middleware**: Tambahkan `/toolbox` ke array `protectedPaths` di `middleware.ts` yang sudah ada.

8. **SEO**: Metadata halaman — title: "Vibe Coding Toolbox — AIGYPT", description sesuai.
