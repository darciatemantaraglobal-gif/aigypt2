### Format Card Testimoni (placeholder, instruksi untuk pemilik website)

Buat 3-4 card testimoni dalam layout grid atau carousel horizontal. Gunakan struktur data berikut sebagai placeholder yang mudah diedit (simpan di file terpisah `lib/testimonials.ts` agar pemilik website bisa mengganti tanpa menyentuh komponen):

```typescript
export interface Testimonial {
  quote: string        // kutipan spesifik, bukan generik
  result: string        // hasil konkret, contoh: "Hemat 5 jam per minggu"
  name: string          // nama depan plus inisial, contoh: "Ahmad R."
  role: string           // contoh: "Mahasiswa Syariah, Tahun ke-3"
}

export const testimonials: Testimonial[] = [
  {
    quote: "GANTI_DENGAN_KUTIPAN_ASLI — buat spesifik, sebutkan masalah dan hasil konkret",
    result: "GANTI_DENGAN_HASIL_KONKRET",
    name: "GANTI_NAMA",
    role: "GANTI_PERAN"
  }
  // tambahkan 2-3 lagi dengan pola yang sama
]
```

Tampilkan placeholder ini di production dengan watermark kecil "Contoh testimoni, akan diperbarui" agar tidak terkesan menipu sebelum testimoni asli dimasukkan oleh pemilik. Tambahkan juga komentar di kode:
```typescript
// TODO: Ganti dengan testimoni asli dari peserta demo/beta AIGYPT.
// Format yang baik: masalah spesifik, plus hasil terukur, plus nama dan peran.
```

Styling card: glassmorphism, tanda kutip besar dekoratif (ungu muted) di pojok, nama dan peran di bawah dengan border-top hairline tipis.

## DETAIL ANIMASI TAMBAHAN

1. Tab Showcase: transisi konten antar tab dengan fade plus slight slide (Framer Motion AnimatePresence), durasi sekitar 250ms
2. Kolom Sebelum/Sesudah: kolom "Dengan AI" punya subtle glow pulse di border untuk menarik perhatian
3. Badge waktu: muncul dengan scale-in kecil setelah konten utama tampil, delay sekitar 150ms, terasa seperti stempel
4. Grid ringkas showcase: stagger reveal saat scroll, tiap card muncul berurutan dengan delay kecil
5. Card testimoni: hover halus, tanda kutip dekoratif punya opacity rendah yang sedikit menguat saat hover
6. Stat grid di Keanggotaan: angka melakukan count-up animation singkat saat masuk viewport, dari 0 ke angka final, durasi sekitar 800ms
7. Hormati prefers-reduced-motion di semua animasi baru
