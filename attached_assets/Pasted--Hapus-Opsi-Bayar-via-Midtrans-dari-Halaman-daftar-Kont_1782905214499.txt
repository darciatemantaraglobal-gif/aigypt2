# Hapus Opsi Bayar via Midtrans dari Halaman /daftar

## Konteks

Halaman `/daftar` saat ini punya dua tombol pembayaran: **Bayar via Midtrans** (primary) dan **Bayar via QRIS** (secondary). Midtrans perlu dihapus sementara — tinggalkan hanya QRIS.

Semua perubahan hanya di satu file: `artifacts/aigypt/src/pages/daftar.tsx`.

---

## Perubahan 1 — Hapus useEffect Loader Script Midtrans

Cari dan hapus seluruh blok `useEffect` berikut (dari baris pertama hingga `}, []);`):

```tsx
useEffect(() => {
  const isProduction = import.meta.env.VITE_MIDTRANS_IS_PRODUCTION === "true";
  const clientKey = import.meta.env.VITE_MIDTRANS_CLIENT_KEY ?? "";
  const snapUrl = isProduction
    ? "https://app.midtrans.com/snap/snap.js"
    : "https://app.sandbox.midtrans.com/snap/snap.js";

  const existing = document.getElementById("midtrans-snap");
  if (existing) {
    setSnapReady(true);
    return;
  }

  const script = document.createElement("script");
  script.id = "midtrans-snap";
  script.src = snapUrl;
  script.setAttribute("data-client-key", clientKey);
  script.onload = () => setSnapReady(true);
  script.onerror = () => console.error("[Snap] Gagal memuat Midtrans Snap.js");
  document.head.appendChild(script);
  scriptRef.current = script;

  return () => {
    if (scriptRef.current && document.head.contains(scriptRef.current)) {
      document.head.removeChild(scriptRef.current);
    }
  };
}, []);
```

## Perubahan 4 — Hapus Tombol "Bayar via Midtrans" dan Divider "atau"

Di dalam JSX, cari blok `<div className="pt-2 space-y-3">`. Di dalamnya ada:
1. Tombol primary "Bayar via Midtrans" (`type="submit"`)
2. Divider "atau"
3. Tombol secondary "Bayar via QRIS" (`type="button"`)

**Hapus hanya item 1 dan 2**, yaitu:

```tsx
<button
  type="submit"
  disabled={loading}
  className="w-full inline-flex items-center justify-center gap-2 text-base font-medium text-white transition-all duration-200 disabled:opacity-60 disabled:cursor-not-allowed"
  style={{
    background: "#7C3AED",
    borderRadius: "12px",
    padding: "15px 24px",
    minHeight: "52px",
    boxShadow: "0px 4px 16px rgba(124,58,237,0.35)",
  }}
>
  {loading ? (
    <><IconSpin />Memproses...</>
  ) : (
    <>Bayar via Midtrans <IconArrow /></>
  )}
</button>

<div className="flex items-center gap-3">
  <div className="flex-1 h-px" style={{ background: "rgba(255,255,255,0.06)" }} />
  <span className="text-xs font-mono" style={{ color: "#3F3F46" }}>atau</span>
  <div className="flex-1 h-px" style={{ background: "rgba(255,255,255,0.06)" }} />
</div>
```
