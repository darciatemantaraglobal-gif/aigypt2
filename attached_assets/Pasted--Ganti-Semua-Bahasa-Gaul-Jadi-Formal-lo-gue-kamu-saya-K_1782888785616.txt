# Ganti Semua Bahasa Gaul Jadi Formal — "lo/gue" → "kamu/saya"

## Konteks

Di seluruh teks user-facing website AIGYPT, banyak penggunaan bahasa gaul Jakarta seperti "lo", "gue", "kalo", "ga" yang perlu diganti ke bahasa yang lebih sopan dan inklusif: "kamu", "saya/kami", "kalau", "tidak/nggak".

Total ada ~150 kemunculan tersebar di 10 file.

---

## Aturan Penggantian Global

Terapkan aturan ini secara konsisten di **semua file yang disebutkan**:

| Kata Asli | Ganti Dengan | Catatan |
|---|---|---|
| `lo` (kata ganti orang ke-2) | `kamu` | Hanya yang bermakna "kamu", bukan bagian kata lain |
| `gue` | `saya` | Saat merujuk narrator/platform |
| `kalo` | `kalau` | |
| ` ga ` / `ga ` | `tidak` atau `nggak` | Sesuaikan konteks kalimat |
| `udah` | `sudah` | |
| `ngerti` | `mengerti` | |
| `bikin` | `membuat` | Hanya di teks narasi, bukan di nama variabel/fungsi |
| `buat lo` | `untuk kamu` | |
| `buat gue` | `untuk saya` | |

**PENTING:** Hanya ubah teks yang tampil ke user (string di dalam JSX, string konten, label). Jangan ubah nama variabel, nama fungsi, nama kelas CSS, komentar kode, atau logika apapun.

## File 10: `artifacts/aigypt/src/lib/materiContent.ts`

File terbesar — 87 kemunculan. Terapkan find & replace global dengan aturan yang sama:

Lakukan **find & replace** berikut secara berurutan di file ini:

1. `" lo "` → `" kamu "` (dengan spasi di kedua sisi untuk menghindari false positive)
2. `" lo.` → `" kamu.`
3. `" lo,` → `" kamu,`
4. `" lo:` → `" kamu:`
5. `" lo'` → `" kamu'`
6. `(lo ` → `(kamu `  
7. `lo)` → `kamu)`
8. `\nlo ` → `\nkamu `
9. `gue` → `saya`
10. `kalo` → `kalau`
11. `" ga "` → `" tidak "`
12. `"ga kasih` → `"tidak memberi`
13. `ngerti` → `mengerti`
14. `udah` → `sudah`

Setelah replace global, baca ulang setiap kalimat yang diubah untuk memastikan tidak ada yang janggal secara tata bahasa.

## Yang TIDAK Boleh Diubah

- Nama variabel, fungsi, tipe TypeScript (contoh: `setLocation`, `kelasId`, `setModalKelas`)
- Class names Tailwind (contoh: `"text-[#94A3B8]"`)
- Komentar kode (`// ...`)
- Import statements
- Logika kondisional apapun
- Kata "lo" yang merupakan bagian dari kata lain (contoh: `"color"`, `"below"`, `"follow"`, `"overflow"`, `"dialog"`, `"boolean"`, `"modal"`)
- Nama brand atau proper noun
